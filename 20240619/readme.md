[Bootstrap](https://getbootstrap.com/)

## 優點

- 完整度高
- 學習曲線低
- 可擴充 (SCSS)
- 官方文件齊全
- RWD 控制方便
- 透過 class 組合，簡單的需求甚至不需要自己寫 css
- 不需要編譯器編譯 (相對於 Tailwind CSS)

## 缺點

- 只能滿足 90% 需求
- component 構成複雜（善用文件）
- 眾多 !important，覆寫彈性降低
- 大量 class 組合，主體辨識度較低
- 不支援 IE 11 以下瀏覽器

## 引用

- 下載壓縮檔案
- copy css/bootstrap.min.css
- copy js/bootstrap.min.js

`bs5.html`
